# 🏆 SPORTIVO - Premium Sportswear E-commerce Website

SPORTIVO is a full-stack sportswear e-commerce web application built using **Node.js, Express.js, MongoDB, EJS, and Tailwind CSS**. It provides a seamless shopping experience with secure authentication, product browsing, cart management, and an intuitive admin interface.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* Secure Authentication
* Browse Sports Products
* Product Details Page
* Add to Cart
* Remove from Cart
* Responsive User Interface
* Modern Tailwind CSS Design

### 🛠️ Admin Features

* Add New Products
* Manage Products
* Upload Product Images
* View Product Listings
* Update Store Inventory

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Kapils17/Sportivo.git
```

### 2. Navigate to the project

```bash
cd Sportivo
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file in the project root.

```env
PORT=3000
MONGODB_URI=your_mongodb_connection_string
JWT_KEY=your_jwt_secret
EXPRESS_SESSION_SECRET=your_session_secret
```

### 5. Start the development server

```bash
npm start
```

or

```bash
node app.js
```

The application will be available at:

```text
http://localhost:3000
```
